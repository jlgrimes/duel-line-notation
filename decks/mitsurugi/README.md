# Mitsurugi

This deck is DLN's first language fixture because its lines exercise several important concepts at once:

- tributing as an activation cost
- effects triggered by being Tributed
- Special Summons performed during another effect's resolution
- simultaneous triggers and explicit Chain ordering
- Ritual materials selected from multiple zones
- recursion that depends on whether a Ritual Monster was properly summoned
